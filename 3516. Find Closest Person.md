3516. Find Closest Person

```js
/**
 * @param {number} x
 * @param {number} y
 * @param {number} z
 * @return {number}
 */
var findClosest = function(x, y, z) {

    if(Math.abs(z - x) < Math.abs(z - y))
        return 1
    else if(Math.abs(z - x) > Math.abs(z - y))
        return 2
    else
        return 0
    
};
```
```py
class Solution:
    def findClosest(self, x: int, y: int, z: int) -> int:
        
        if abs(z - x) < abs(z - y):
            return 1
        elif abs(z - x) > abs(z - y):
            return 2
        else:
            return 0
```
