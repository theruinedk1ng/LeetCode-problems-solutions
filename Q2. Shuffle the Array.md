Q2. Shuffle the Array

```js
/**
 * @param {number[]} nums
 * @param {number} n
 * @return {number[]}
 */
var shuffle = function(nums, n) {
    let answer = new Array(2 * n);

    for (let i = 0; i < 2 * n; i ++) {
        if (i % 2 === 0)
            answer[i] = nums[Math.floor(i / 2)];
        else
            answer[i] = nums[Math.floor(i / 2) + n];
    };

    return answer;
};
```
```py
class Solution:
    def shuffle(self, nums: List[int], n: int) -> List[int]:
        
        return [nums[i // 2] if i % 2 == 0 else nums[i // 2 + n] for i in range(2 * n)]
```
