Q3. Find All Numbers Disappeared in an Array

```js
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var findDisappearedNumbers = function(nums) {
    const n = nums.length;
    let seen = new Array(n + 1).fill(false);

    for(let i = 0;i < n;i++)
        seen[nums[i]] = true;

    let result = [];
    for(let i = 1;i <= n;i++)
        if(seen[i] === false)
            result.push(i);

    return result;
};
```
```py
class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        return list(set(range(1, len(nums) + 1)) - set(nums))
```
