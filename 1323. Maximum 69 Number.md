1323. Maximum 69 Number

```py
class Solution:
    def maximum69Number (self, num: int) -> int:

        result = num
        num = list(str(num))
        n = len(num)

        for i in range(n):
            if num[i] == '9':
                continue
            else:
                result = max(result, int(''.join(num[:i]) + '9' + ''.join(num[i + 1:])))
        
        return result
```
